# Module Ownership

| Module | Owns | Must not own |
|---|---|---|
| mt5_gateway | fresh market/account/indicator evidence | scenario reasoning |
| snapshot | synchronized immutable analysis input | trading signals |
| perception | facts, features, detected events | permission |
| fusion | coherent market state and conflicts | order placement |
| scenario | possible next paths, invalidation, expiry | guaranteed prediction |
| entry | limit/early/confirmed/continuation candidates | final authorization |
| safety | freshness, integrity, contradictions, hard gates | market interpretation |
| watcher | event detection and job triggers | AI reasoning every tick |
| training | blind replay, scoring, drills | production policy mutation |
| knowledge | episodes, hypotheses, validated findings | silent rule promotion |
| Codex skills | workflow selection, analysis, audit, reports | modifying raw evidence/checker outputs |
