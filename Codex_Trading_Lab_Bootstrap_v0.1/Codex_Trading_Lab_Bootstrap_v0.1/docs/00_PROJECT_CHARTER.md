# Project Charter

## Mission
Create a standalone agentic trading research and decision platform whose purpose is not decorative market analysis but practical, sharp, timely entry planning with enough opportunity throughput to be useful.

## System formula
Fresh Evidence → Perception → Market State → Scenario Tree → Entry Candidates → Permission Review → Manual Execution → Review → Learning

## Success definition
The system should improve decision quality, reduce avoidable bad entries, preserve usable trade frequency, and make every conclusion auditable.

## Non-goals for MVP
- Automatic order execution
- Self-modifying production rules
- Full ICT/SMC coverage on day one
- Multi-symbol production
- Cloud deployment
- Fabricated probability estimates
