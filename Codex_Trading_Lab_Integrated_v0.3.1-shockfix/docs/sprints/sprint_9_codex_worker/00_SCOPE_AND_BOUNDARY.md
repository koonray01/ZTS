# Scope and Boundary

## Included
- queue consumption
- leasing and recovery
- skill resolution
- model-provider abstraction
- allowlisted tool loop
- result validation
- retries/dead-letter
- result/audit persistence

## Excluded
- real model credentials
- real provider network calls
- MT5 connection
- broker execution
- automatic policy or skill deployment
- background daemon/service manager
