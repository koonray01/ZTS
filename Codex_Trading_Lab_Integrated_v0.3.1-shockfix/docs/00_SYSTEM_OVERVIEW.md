# Integrated System Overview

Codex Trading Lab is a standalone decision laboratory for XAUUSD. It separates
perception, scenario construction, entry evaluation, deterministic permission,
live monitoring, replay learning and AI orchestration.

```text
Data Source
→ Snapshot Contract
→ Basic/Advanced Eyes
→ Market Fusion
→ Scenario Tree
→ Entry Candidates
→ Part 3 Permission
→ Manual Execution Proposal
→ Live Runtime / Position Monitor
→ Replay Episodes
→ Knowledge Promotion Gates
→ Codex Worker
```

The AI worker coordinates tools and explains state. It is not the source of truth,
cannot create permission and cannot execute trades.
